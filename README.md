# 💊 セルフメディケーション税制・レシート管理ツール

スマホのGeminiアプリでレシートを読み取り、セルフメディケーション税制対象商品の購入額を自動で集計・管理するツールです。

## 🌟 特徴
- **GeminiによるOCR/抽出**: レシート画像から店舗名、日付、商品名、対象金額を精度高く抽出。
- **重複防止機能**: すでにインポートしたファイルや、同一商品の二重登録を自動でスキップ。
- **店舗別集計**: 確定申告時にそのまま記入できるよう、店舗ごとに合計金額を出力。

## 🚀 使い方

### 1. GeminiでレシートをCSV化
スマートフォンでレシートを撮影し、Geminiアプリに以下のプロンプト（例）と共に送信します。
> 「このレシートから、購入店舗名,購入年月日,商品名,単品税抜価格,価格,個数,セルフメディケーション対象金額を抽出し、CSV形式で出力してください。」

得られたテキストを `.csv` ファイルとして `csv_files/` フォルダに保存します。

### 2. データのインポート
Macのターミナルで以下のコマンドを実行し、SQLiteデータベースへ取り込みます。
